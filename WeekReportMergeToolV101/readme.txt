
XLSX merge tool V1.1

This tool can collect all Weekly xlsx and merge sheet into one xlsx.

Usage:
1. open exe at: \CSexcel\CSexcel\bin\Debug\CSexcel.exe
2. select Weekly Report root directly such as "ww2016"
3. enter a sheet name, such as "Customer Support"
4. click Merge, the finial output file is under same folder of .exe.

Limitation:
1. only support xlsx.
2. sheet name can be set by textbox.
3. use VS2010.
4. merge xlsx can be found at application(*.exe) path
5. exe is depend on DLL in the same dir, do not move .exe to other places.
